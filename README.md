# Spring-Boot_Baseline

---------

### Git 로컬 저장소 생성
```
> git init
> git add -A / git add 파일명
> git commit -m "(commit message)"
```

로컬 git 저장소를 온라인 사이트와 연결
```
> git remote add origin https://github.com/계정이름/저장소이름.git
> git push -u origin master
```

변경 사항 반영
```
> git add -A
> git commit -m "(commit message)"
> git push -u origin master
```
